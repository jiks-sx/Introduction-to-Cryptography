# Real-World Cryptography Usage

## HTTPS
- Certificate-based authentication
- Secure key exchange
- Encrypted data transfer

process :
- Client connects
- Server sends certificate
- Client verifies CA
- Public key exchange
- Symmetric session key created
- Encrypted communication begins

## VPN
- Encrypts network traffic
- Ensures confidentiality and integrity

VPN provides:
- Confidentiality
- Integrity
- Authentication

Uses:
- AES for encryption
- RSA/ECDH for key exchange
- HMAC for integrity
